### マイグレーション実行

1. ダッシュボード → **SQL Editor** → **New query**
2. `supabase/migrations/20250303000000_initial_schema.sql` の内容を貼り付け
3. **Run** をクリック
4. 初期テナントを1件登録: `INSERT INTO tenants (name) VALUES ('テラスターファーマシー');`

- チーム/プロジェクト: logicworks-projects

## デプロイ手順（初回）

1. GitHub に `terastar-LINE` リポジトリを作成
2. ローカルで `git remote add origin` → `git push -u origin main`
3. Vercel → Add New → Project → 該当リポジトリをインポート
4. 環境変数 `LINE_CHANNEL_SECRET`、`LINE_CHANNEL_ACCESS_TOKEN` を設定
5. デプロイ完了後、[Webhook URL登録手順](Webhook_URL登録手順.md) を実行
