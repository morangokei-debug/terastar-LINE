# リッチメニュー設定

LINE公式アカウントのリッチメニュー（画面下部のメニュー）を設定する手順です。

## メニュー構成

| ボタン | 動作 |
|--------|------|
| 処方箋送信 | `/prescription-submit` を開く（患者が名前・メモを送信） |
| ホームページ | 薬局のホームページへ誘導 |
| メッセージ入力 | 「お問い合わせ」を送信し、チャットでメッセージ入力 |

## 設定手順

### 1. 環境変数

`.env.local` に以下を設定（任意）:

- `NEXT_PUBLIC_PHARMACY_HOMEPAGE_URL` … ホームページのURL（未設定時はアプリのURL）

### 2. マイグレーション

処方箋送信リクエスト用テーブルを追加:

```bash
npx supabase db push
# または Supabase ダッシュボードで 20250304000000_prescription_requests.sql を実行
```

### 3. リッチメニューを設定

1. 薬剤師アカウントでログイン
2. **設定** → **リッチメニューを設定する** をクリック
3. 成功メッセージが表示されれば完了

※ `LINE_CHANNEL_ACCESS_TOKEN` が設定されている必要があります。

### 4. 処方箋リクエストの確認

患者が「処方箋送信」から送信した内容は **処方箋リクエスト** 画面で確認できます。
