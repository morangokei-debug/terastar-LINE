# テラスターファーマシー LINEフォローアップ 操作マニュアル

**対象者：** 薬局スタッフ（非エンジニア）  
**本番URL：** https://terastar-line.vercel.app  
**最終更新：** 2026年3月

---

## このシステムでできること

| できること | 説明 |
|-----------|------|
| 患者管理 | LINE友だち追加で自動登録。名前・生年月日は患者または薬局側で登録可能 |
| フォローアップ | 薬渡し後に自動で体調確認メッセージを送信。血圧・症状の自由入力も対応 |
| 処方箋受信 | 患者がLINEから処方箋写真を送信。受信時に薬局へ通知 |
| チャット | 患者とのLINEメッセージをダッシュボードで確認・返信 |
| 一斉送信 | 友だち全員または選択した患者にメッセージを一括送信 |

## STEP 01　ログイン

❶ ブラウザで本番URL（https://terastar-line.vercel.app）を開く  

❷ 「ログイン」または「ログインする」をクリック  

❸ メールアドレスとパスワードを入力し、「ログイン」をクリック  

❹ ログインに成功すると、ダッシュボード（トップ画面）に遷移します  

> 💡 ポイント！ 初回利用時は「新規登録」からアカウントを作成してください。

> ⚠️ 注意！ パスワードを忘れた場合は、ログイン画面の「パスワードを忘れた場合」から再設定できます。

---

## STEP 02　画面の見かた

ログイン後の画面は、**左側のメニュー**と**右側のメインエリア**で構成されています。

### 左メニュー（サイドバー）

| メニュー名 | 説明 |
|-----------|------|
| ダッシュボード | トップ画面。患者数・フォロー予定数などを表示 |
| 患者一覧 | LINEで友だち追加された患者の一覧 |
| フォローアップ文例・登録 | 体調確認メッセージのテンプレート（文例）を登録 |
| フォローアップ実施入力 | 薬渡し日を入力し、フォローアップ送信予定を登録 |
| フォローアップ返信内容確認 | 患者からの返信（体調・血圧など）を確認 |
| チャット | 患者とのLINEメッセージ履歴 |
| 一斉送信 | 友だち全員や選択した患者にメッセージを一括送信 |
| 受信処方箋 | LINEから送信された処方箋の一覧 |
| 設定 | LINE連携・リッチメニュー・通知先などの設定 |

### 右側メインエリア

選択したメニューに対応する画面が表示されます。

## STEP 03　患者一覧・名前登録

### 患者一覧の確認

❶ 左メニュー「患者一覧」をクリック  

❷ 患者の一覧が表示されます。各列の意味は以下の通りです。

- **患者名** … 登録されている名前（LINE友だち追加直後は「LINE友だち追加」や「LINE（日付）」と表示）
- **生年月日** … 登録されている場合に表示
- **LINE** … 「紐付け済」＝LINEと連携済み、「未紐付け」＝手動登録でLINE未連携
- **登録日** … 患者が登録された日付

### 薬局側から名前・生年月日を登録する

患者さんが自分で登録していない場合、薬局側から登録できます。

❶ 患者一覧で、名前をクリック（患者詳細画面へ）  

❷ 「基本情報」のフォームで、**患者名**と**生年月日**を入力  

❸ 「保存する」をクリック  

> 💡 ポイント！ LINEで友だち追加された方の名前・生年月日は、患者さん自身が登録フォームから入力するか、薬局側で患者詳細画面から登録・編集できます。

### 患者を新規登録する（手動）

❶ 「新規登録」ボタンをクリック  

❷ 患者名・生年月日・電話番号を入力  

❸ 「登録する」をクリック  

❹ 後からLINEと紐付ける場合は、患者詳細画面の「LINE紐付け」で、友だち追加済みのLINEユーザーを選択して紐付けます。

## STEP 04　フォローアップ文例・登録

フォローアップメッセージの**テンプレート（文例）**を登録します。ここで登録した文例を、STEP 05の「フォローアップ実施入力」で選択して使います。

### 新規登録

❶ 左メニュー「フォローアップ文例・登録」をクリック  

❷ 「新規登録」ボタンをクリック  

❸ 以下の項目を入力します。

| 項目 | 説明 | 例 |
|------|------|-----|
| パターン名 | 文例の名前（管理用） | 抗生物質、降圧薬変更 |
| 送信日数 | 薬渡しから何日後に送るか | 3、5 |
| メッセージ文例 | 患者に送るメッセージ本文 | 下記参照 |
| 返信選択肢 | 患者が選べるボタン（カンマ区切り） | とても良い,良い,普通,悪い,とても悪い |
| 自由入力欄 | 血圧・症状などを自由入力してもらう場合にON | チェックを入れる |
| 自由入力の案内文 | 自由入力時の案内文 | 気になる症状や伝えたいことがあればご記入ください。 |
| 返信後のお礼メッセージ | 患者が返信した後に自動で送るメッセージ | ご回答ありがとうございます。… |

❹ 「登録する」をクリック  

### メッセージ文例の変数

文例には以下の**変数**が使えます。送信時に自動で置き換わります。

| 変数 | 置き換えられる内容 |
|------|-------------------|
| {患者名} | 患者の登録名 |
| {薬名} | フォローアップ実施入力で入力した薬名（カンマ区切りで複数可） |
| {日数} | 送信日数（パターンで設定した値） |

**例：**  
`{患者名}様、{薬名}をお渡ししてから{日数}日経ちました。体調はいかがですか？`  
→ 「山田太郎様、アムロジピン錠5mgをお渡ししてから5日経ちました。体調はいかがですか？」

### 編集・削除

一覧の「編集」をクリックすると、既存の文例を編集できます。

## STEP 05　フォローアップ実施入力

実際に**薬を渡した日**を入力し、フォローアップメッセージの送信予定を登録します。

❶ 左メニュー「フォローアップ実施入力」をクリック  

❷ 以下の項目を入力します。

| 項目 | 説明 |
|------|------|
| 患者 | 薬を渡した患者を選択（必須） |
| 薬名 | 渡した薬の名前。カンマ区切りで複数入力可（任意） |
| フォロー日程パターン | 使用する文例を選択（必須） |

❸ 「登録する」をクリック  

❹ 登録後、「送信予定一覧」で予定を確認できます。

> 💡 ポイント！ パターンで設定した「送信日数」後（例：3日後）の朝9時頃に、Cron（自動実行）でメッセージが送信されます。すぐに送りたい場合は、STEP 06の「今すぐ送信」を使います。

---

## STEP 06　フォローアップ送信予定

登録したフォローアップの送信予定を確認・編集できます。

❶ 左メニュー「フォローアップ実施入力」→「送信予定一覧」をクリック  
　または「フォローアップ実施入力」画面の「送信予定一覧」リンクをクリック  

❷ 一覧が表示されます。

| 列 | 説明 |
|----|------|
| 患者名 | 対象患者 |
| 送信予定日時 | 送信予定の日時 |
| パターン | 使用する文例名 |
| 薬名 | 登録した薬名 |
| 状態 | 未送信 / 送信済み |

### 送信予定日時を編集する

❶ 該当行の「編集」ボタンをクリック  

❷ 日時を変更し、「保存」をクリック  

### 今すぐ送信する

送信予定日を待たずに、すぐにLINEでメッセージを送りたい場合：

❶ 該当行の「今すぐ送信」ボタンをクリック  

❷ 確認ダイアログで「OK」をクリック  

❸ 患者のLINEにメッセージが送信されます。

> ⚠️ 注意！ 「今すぐ送信」は未送信の予定にのみ表示されます。送信済みの場合は表示されません。また、患者にLINEアカウントが紐付いていない場合は送信できません。

---

## STEP 07　フォローアップ返信内容確認

患者がフォローアップメッセージに返信した内容を確認します。

❶ 左メニュー「フォローアップ返信内容確認」をクリック  

❷ 返信一覧が表示されます。患者名・返信日時・返信内容（選択肢または自由入力テキスト）を確認できます。

> 💡 ポイント！ 患者が「とても良い」などのボタンを押した場合も、血圧値や症状をテキストで入力した場合も、ここに記録されます。

---

## STEP 08　受信処方箋

患者がLINEのリッチメニュー「処方箋送信」から送信した処方箋の一覧です。

❶ 左メニュー「受信処方箋」をクリック  

❷ 一覧が表示されます。

| 列 | 説明 |
|----|------|
| お名前 | 患者が入力した名前 |
| 生年月日 | 患者が入力した生年月日 |
| メモ | 患者が入力したメモ |
| 処方箋 | 画像がある場合「画像を表示」リンク |
| 送信日時 | 送信された日時 |

> 💡 ポイント！ 処方箋が届くと、設定で登録したLINEアカウントに通知が届きます。見落とし防止にご利用ください。

---

## STEP 09　チャット

患者とのLINEメッセージ履歴をダッシュボードで確認・返信できます。

❶ 左メニュー「チャット」をクリック  

❷ 患者一覧が表示されます。未読がある場合は「要対応」タブに表示されます。  

❸ 患者名をクリックすると、メッセージ履歴が表示されます。  

❹ 返信する場合は、画面下部の入力欄にメッセージを入力し、送信ボタンをクリックします。

> ⚠️ 注意！ チャット機能を使うには、LINE Official Account Manager の応答設定で「応答メッセージ」をオフにする必要があります。

---

## STEP 10　一斉送信

友だち全員または選択した患者に、メッセージを一括送信できます。

❶ 左メニュー「一斉送信」をクリック  

❷ 「送信先」で以下を選択します。

- **友だち全員** … LINE公式アカウントを友だち追加した全員に送信
- **患者を選択** … 一覧から送信先の患者を選択

❸ 「メッセージ」欄に送りたい内容を入力  

❹ 「送信内容を確認する」をクリック  

❺ 確認画面で内容を確認し、送信を実行します。

> ⚠️ 注意！ 送信人数分がLINEの料金対象になります。送信前に必ず内容を確認してください。

## STEP 11　設定

### 処方箋受信時のLINE通知

処方箋が届いたときに、登録したLINEアカウントに通知を送ります。

❶ 「登録用コードを発行」をクリック  

❷ 表示された6文字のコードをメモ  

❸ **スマホのLINEアプリ**で、薬局のLINE公式アカウントを開く（友だち追加済みであること）  

❹ トークに「通知登録 XXXXXX」（コードを入力）と送信  

❺ 「通知先として登録しました」と返信されれば完了  

> ⚠️ 注意！ ダッシュボードのチャットやLINE公式アカウント管理画面から送っても登録できません。必ず**スマホのLINEアプリ**のトーク画面から送信してください。コードは15分で期限切れになります。

### リッチメニュー設定

LINEの画面下部に表示される「処方箋送信」「ホームページ」などのボタン（リッチメニュー）を設定します。

❶ 「リッチメニューを設定する」ボタンをクリック  

❷ 設定が完了するまで待ちます。初回のみ実行してください。  

❸ 友だち追加後にリッチメニューが表示されない場合は、このボタンを押した後、該当の方に一度友だち削除→再追加してもらってください。

### 友だち追加時の挨拶文

環境変数 `LINE_WELCOME_MESSAGE` で挨拶文をカスタマイズできます。  
LINE Official Account Manager の「あいさつメッセージ」は**オフ**にしてください。オンのままだと当システムの挨拶が届きません。

### タブレットでフルスクリーン表示

タブレット・スマホで開いた時、画面右下にフルスクリーンボタンが表示されます。タップするとアドレスバーなどを非表示にできます。  
「ホーム画面に追加」でアプリのように起動すると、起動時からフルスクリーンで開きます。

## よくある質問

### Q1. 患者一覧に名前が「LINE友だち追加」のままです。どうすればいいですか？

**A.** 以下のいずれかで登録できます。

- **患者さんに登録してもらう** … 友だち追加時に送られるURLから、お名前・生年月日を入力してもらう
- **薬局側で登録する** … 患者一覧で該当患者をクリック→患者詳細の「基本情報」で名前・生年月日を入力→保存

### Q2. フォローアップメッセージが届かないようです。

**A.** 以下を確認してください。

1. 患者にLINEアカウントが紐付いているか（患者一覧で「紐付け済」になっているか）
2. 送信予定日になっているか（Cronは毎日9時頃に実行されます）
3. すぐに送りたい場合は「今すぐ送信」ボタンを使用してください

### Q3. 処方箋が届いたことを知りたいです。

**A.** 設定画面の「処方箋受信時のLINE通知」で、通知先を登録してください。処方箋が届くと、登録したLINEアカウントに通知が届きます。

### Q4. リッチメニューが表示されません。

**A.** 設定画面の「リッチメニューを設定する」ボタンを押してください。その後、該当の方に一度友だち削除→再追加してもらうと表示されます。

### Q5. 友だち追加時の挨拶が出ません。

**A.** LINE Official Account Manager の「応答設定」で「あいさつメッセージ」を**オフ**にしてください。また、Webhookの利用が有効になっているか、LINE Developers コンソールで確認してください。
